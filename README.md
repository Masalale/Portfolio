# Portfolio

Personal portfolio built with Next.js 16, TypeScript, and Tailwind CSS.

## Quick Start

```bash
bun install
bun dev
```

Open http://localhost:3000

## Tech Stack

- Next.js 16 + TypeScript
- Tailwind CSS v4
- GSAP + Lenis (animations & smooth scroll)
- Lucide React (icons)

## Structure

- `app/` - pages and global styles
- `components/` - layout, sections, UI components
- `data/` - content (projects, skills, experience, social)
- `hooks/` - scroll animations & navigation
- `utils/` - shared utilities

## Deploy

```bash
bun run build
bun start
```

Or push to GitHub and deploy on Vercel.
